Project Name: Hero IO – App House (Web Application)

Description: 
A responsive web application that allows users to explore apps, view detailed information, and manage installations with persistent local storage support.

Features: 
1. Developed a responsive web application to browse trending and all apps with a clean card-based UI.
2. Implemented dynamic routing to display individual app details pages with full information.
3. Integrated an app installation system using LocalStorage with real-time UI updates (Install → Installed → Disabled).
4. Built an installation management page allowing users to view and uninstall apps with persistent data handling.

Technologies Used:
React.js
React Router
Tailwind CSS & DaisyUi
JavaScript (ES6+)
LocalStorage
Axios
Netlify (Deployment)
